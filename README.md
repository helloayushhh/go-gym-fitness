# gogym

a responsive fitness website built while learning frontend development.

---

## why i built this

this was one of the projects that helped me understand how websites are built from scratch.

i used it to practice responsive layouts, reusable components and creating a clean user experience using only frontend technologies.

---

## what's inside

- responsive landing page
- modern fitness-themed design
- reusable sections
- smooth navigation
- bootstrap components

---

## built with

- html
- css
- javascript
- bootstrap

## run locally

```bash
git clone https://github.com/helloayushhh/GoGym-fitness.git
cd GoGym-fitness
```

open `index.html` in your browser or use a local server.
